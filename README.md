#bunny.tamagotchi

This project is a simple tamagotchi game where you take care of a bunny. It was made using java for both the code and the UI. I made this game because wanted to practice java coding, and I specifically used java swing for 
the graphics because I liked the "retro" style. Through this project, I learned how to use java swing, which was also one of the more diffucult parts of the project, since the placement of components was a bit finicky at times. I also got to practice using java methods and classes. Another difficult part was figuring out how to deploy the java app as a web application, and I ended up using Render. Overall, this game was a useful project for learning java GUIs.
